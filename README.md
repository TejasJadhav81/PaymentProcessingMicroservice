# Payment Processing Microservice

A Spring Boot microservice for handling payment processing operations.

## Tech Stack

- Java 17
- Spring Boot 3.1.x
- Maven
- Lombok
- Gson
- ModelMapper

## Getting Started

### Prerequisites

- Java 17+
- Maven 3.6+

### Run Locally

```bash
mvn spring-boot:run -Plocal
```

The service starts on **port 8082**.
